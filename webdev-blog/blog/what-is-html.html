What is HTML? (My Beginner Guide)

Published on June 16, 2025 • By Roshan

When I started learning web development, the first thing I came across was something called HTML. At first, it looked weird — just a bunch of angled brackets and tags. But once I understood the basics, it started to make sense.

💡 So... What is HTML?

HTML stands for HyperText Markup Language. It’s not a programming language — it’s a markup language that’s used to define the structure of web pages. Think of it like building blocks: you use HTML to say “this is a heading,” “this is a paragraph,” “this is a link,” etc.

🔨 Why Do We Use HTML?

Every website you visit — no matter how cool or modern it looks — starts with HTML. It’s the skeleton. Without it, your browser wouldn’t know what to show you.

👀 Basic Example

Here’s a super basic HTML page:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>

If you save that as an `.html` file and open it in your browser, it will work. That’s how simple HTML is.

📄 Some Basic Tags to Remember

  • <h1> to <h6> – Headings
  • <p> – Paragraph
  • <a> – Link
  • <img> – Image
  • <div> – Section/container

✅ Final Thoughts

HTML is where it all begins. Once you’re comfortable with it, you can move on to styling (CSS) and then functionality (JavaScript). In my next post, I’ll break down how CSS works and how it makes your HTML look good.
