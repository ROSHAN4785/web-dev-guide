🎨 CSS Basics

What is CSS? 👗 (Roshan's Simple Guide)

Published: June 16, 2025

You’ve built your first HTML page… but it looks like plain text. 😶 So, how do websites actually look *cool*? That’s where CSS comes in.

🧵 What is CSS?

CSS stands for Cascading Style Sheets. It’s used to style your website — like changing colors, fonts, spacing, layout, and even animations.

🎨 Why Use CSS?

  • Change the background, text color, and fonts
  • Make your layout responsive (works on all devices)
  • Add hover effects, transitions, and animations
  • Make websites look modern and clean

📦 How Do You Use CSS?

You can add CSS in three ways:

  1. Inline: Inside an HTML tag (not recommended)
  2. Internal: In a <style> tag inside HTML
  3. External: Linked via a .css file — best practice ✅

🧪 Quick Example

<style>
  body {
    background-color: #f0f0f0;
    color: #333;
    font-family: sans-serif;
  }
</style>

This simple CSS changes the background, text color, and font of the page. 🎉

💡 My Tip:

Start small. Don’t worry about writing fancy styles. Try changing colors, fonts, and spacing. Play around. That’s how I learned too!
